## EShop

### Description
A simple web app that allow users to view orders, add items to them and pay.

### Getting Started
Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.


### Dependencies
- Next.js
- React
- Tailwind CSS
- Zustand
- Axios
- React Hook Form
- shadcn
